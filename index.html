Squad Mortar Calculator

Instructions

(Click to expand/collapse)
Input the grid and (optional) subgrid coordinates of both the mortar position and the target position. The subgrid part is and is subdivided using keypad keys, and you can do as many subdivisions as you please.

This page will automagically compute a fire solution once all inputs are valid. Internally, this page uses the range tables for mortars provided in-game, linearly interpolates between the values, and warns you if the target is out of range.

Valid Grid Coordinates

  • b2
  • F02
  • C1-12
  • b3-4882111
